Kommander jeg har brukt til oppgave 1.3

1) git clone + url - klonet repositorien som jeg lagde fra github til min maskin
2) echo "melding" > hello.go - opprettet en .go fil
3) ii hello.go - åpner hello.go filen og la til koden
4) git add hello.go - legger til filen klar til commit
5) git commit -m "la til go fil" - legger til en mld og gjør pakker inn endringene klar til push
6) git push origin master - legger ut commiten til github
